# Tienda de Golosinas - William Alfonso

Proyecto demo: sitio web de e-commerce para una tienda de golosinas. Incluye:

- HTML semántico (header, nav, main, section, footer)
- CSS con Bootstrap y estilos propios (styles.css)
- JavaScript que consume una "API REST" local (`/api/products.json`), renderiza productos y maneja un carrito con `localStorage`
- Formulario de contacto que usa Formspree (reemplazá `REEMPLAZA_CON_TU_ID` por tu ID de Formspree)
- Accesibilidad básica (atributos alt, focus visible, navegación por teclado)

## Cómo usar

1. Cloná el repo y ubicá los archivos en la raíz.
2. Reemplazá las imágenes dentro de `/assets` por tus archivos (o usá enlaces externos).
3. Reemplazá el action del formulario con tu ID de Formspree.
4. Probá localmente con `Live Server` o subí a GitHub y activá GitHub Pages o subilo a Netlify.

## Deployment

### GitHub Pages
- Subí el repo a GitHub. En Settings -> Pages, seleccioná la rama `main` y `/ (root)`.

### Netlify
- Arrastrá el folder al panel de Netlify Drop o enlazá el repo para deploy automático.
